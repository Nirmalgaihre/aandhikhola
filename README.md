# 🎓 Annapurna Polytechnic Institute - Management System

A comprehensive, responsive web platform designed to digitize administrative workflows and enhance public engagement for **Annapurna Polytechnic Institute**.

---

## 🏛 Frontend Features

The frontend is designed with a modern **Tailwind CSS** UI, focusing on accessibility and speed.

* **👨‍💼 Principal’s Message:** A dedicated section for official leadership communication and institutional vision.
* **📢 Digital Notice Board:** Categorized updates for exams, admissions, and administration.
* **🖋 Blog Post System:** Engaging articles and news updates regarding campus activities.
* **📄 Publications:** A digital library for research papers, magazines, and journals.
* **🖼 Photo Gallery:** Organized albums showcasing college infrastructure and events.
* **🎥 Video Gallery:** A hub for campus tours and recorded educational events.
* **👥 Staff Directory:** A hierarchical display of faculty and administrative members.

---

## 🛠 Admin Features

* **Content Management:** Full CRUD control over all frontend modules.
* **Hierarchy Reordering:** Drag-and-drop or manual sorting for staff members.
* **ID & Certificate Generator:** One-click automated PDF generation for Student IDs and Academic Certificates.
* **Security:** OTP-based verification for administrative access and password recovery.

---

## 💻 Technical Stack

| Technology | Implementation |
| :--- | :--- |
| **Backend** | PHP 8.4.16 |
| **Framework** | Laravel 13.4.0 |
| **Database** | MySQL (Relational) |
| **Frontend** | Tailwind CSS & Alpine.js |
| **Icons** | FontAwesome 6 |
| **Date Handling** | Carbon API |

---

## 📄 Core Components

* **AnnouncementController:** Manages public programs and short-term course data.
* **StaffController:** Handles staff profiles and organizational ordering.
* **AuthController:** Manages secure logins and OTP validation logic.
